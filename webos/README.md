# NovaDesk Ultimate

NovaDesk Ultimate is an offline browser-based desktop OS built with HTML, CSS and JavaScript.

## Included

• Desktop, taskbar, Start menu and window manager
• Draggable/resizable windows, snapping, taskbar tracking and z-index focus
• Lock screen, quick settings, notifications, widgets and command palette
• File Explorer, Recycle Bin, Text Editor, Browser, Settings and Terminal
• Productivity Hub: Notes, To-Do, Calendar, Calculator, Focus Timer, Unit Converter, Clipboard Board, Password Generator, Paint and Code Editor
• Games Hub: Snake, Pong, Breakout, Space Defender, Flappy Orb, Minesweeper, Memory Match, 2048, Tic Tac Toe, Reaction Test, Number Guess, Word Scramble and Slide Puzzle
• Blockchain Lab: Explorer, Demo Wallet, Mining Simulator, NFT Gallery, Smart Contract Lab, Identity Vault and File Integrity Ledger
• Task Manager, Nova Store, Achievements and System Info
• Persistent localStorage state

## Run locally

Open `index.html` directly, or run:

```bash
python -m http.server 8080
```

Then open:

```text
http://localhost:8080
```

## GitHub Pages

Push all files to the repository root and enable GitHub Pages from the repository settings. `index.html` is the entry point.

## Notes

The blockchain features are local educational simulations only. They do not connect to real networks, use real funds or store real wallet keys.
